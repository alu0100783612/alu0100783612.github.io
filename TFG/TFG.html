Introducción

Las tecnologías de visualización de datos permiten representar información de una forma visual y sencilla. Este hecho es de vital importancia en la era del Big Data, donde se deben de presentar el valor de millones de datos con la mayor precisión y en el menor tiempo posible. En esta propuesta se desarrollará una metodología para la representación visual de datos de un dominio concreto a definir. En este caso trabajaremos en el ámbito educativo.
